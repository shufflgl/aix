# @shufflgl/aix

Agent-oriented AI Extensions Manager CLI for Codex and Claude ext assets.

## Install

```bash
npm install -g @shufflgl/aix
```

## Usage

```bash
aix ext list --json
aix ext install gpt-image-2 --target codex --json
aix secret init gpt-image-2 --backend env
```

Source repository: https://github.com/shufflgl/aix
